# AXIOM v2

Cockpit personal de trading profesional para mercados crypto.

## Visión

Herramienta personal que provee:
- Panorama del régimen de mercado actual
- Gestión de capital adaptada al régimen
- Sugerencias de activos con potencial
- Noticias en tiempo real
- Análisis técnico sobre gráficos de velas
- Watchlist con precios en vivo
- Diario de trades y métricas

## Stack

- Backend: Python 3.11 + FastAPI + PostgreSQL 17
- Frontend: HTML + CSS + Vanilla JS (ES Modules)
- Real-time: Server-Sent Events + polling
- Proxy: Caddy + Cloudflare Tunnel
- Hosting: Notebook server (decentralia)
