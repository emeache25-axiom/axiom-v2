# CODE — AXIOM Quant

Corpus de diseño del sistema cuantitativo de criptomonedas de AXIOM.

CODE construye cada día una **cartera (cocktel)** de pares, cada uno operado
según su naturaleza, cuya suma de retornos netos de costos apunta a ≥1% diario
con riesgo controlado. No predice dirección: selecciona, combina y opera
oportunidades de esperanza matemática positiva.
